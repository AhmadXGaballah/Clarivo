## Product Overview

**Long educational videos are valuable, but your time is even more valuable.**

That is why **Clarivo** exists.

**Clarivo** is an AI summarization application designed to help users extract the essential insights from long-form educational content without committing to the full viewing time. By converting lengthy videos into concise, high-signal summaries, it enables faster learning, better focus, and more efficient knowledge consumption.

Built as a lightweight AI productivity product, Clarivo combines:

- **FLAN-T5** for summarization
- **Whisper** for audio transcription
- **Streamlit** for an intuitive interactive interface

> **Turn hours of content into minutes of clarity.**

## Core Features

### 1. Text Summarization
Paste articles, notes, reports, transcripts, or long-form content and generate concise summaries instantly.

### 2. YouTube Summarization
Provide a YouTube link, transcribe the audio automatically, then generate a readable summary from the transcript.

### 3. Model Modes

Clarivo is designed to support different summarization scenarios:

- **Instant/Recommended Mode** for faster performance and lower compute usage
- **Performance Mode** for the best general-purpose summary quality
- **Long Video Mode** for lengthy educational transcripts
  
### 3. Adjustable Summary Depth
Choose how much detail you want in the output:

- **Short**
- **Medium**
- **Long**
- **Detailed**

### 4. Auto Language Detection
Transcription supports automatic language detection for a smoother input experience.

### 5. Clean Product UI
Clarivo is designed with a focused, minimal interface that keeps the experience simple and user-driven.

### 6. Downloadable Output
Generated summaries can be exported as text for quick reuse and sharing.
